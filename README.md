Grid（无限网格交易程序）
====================

当前状态
-------

已完成第一版工程骨架：WebUI 初始化/登录、配置保存、启动停止占位 bot、实时日志。

快速开始
-------

Windows

- 双击 `scripts/start.bat`
- 浏览器打开 `http://127.0.0.1:8000/`

Linux

- 执行 `bash scripts/start.sh`
- 浏览器打开 `http://127.0.0.1:8000/`

使用说明
-------

- 首次打开会要求设置管理员密码（至少 8 位）
- 登录后可在“交易所配置（Lighter）”里保存参数（当前版本尚未真正下单）
- “运行与状态”里的启动按钮会产生心跳日志，用于验证后台任务与 WebUI 通路
